FIRST Time installing EventON
---------
Installing from wp-admin

Step 1: Go to wp-admin of your website and then Plugins > Add New
Step 2: Unzip the eventON main zip file you downloaded from eventON
Step 3: Upload the eventON main zip file to add new plugin page on your website
Step 4: Once installed activate the plugin

Installing via FTP

Step 1: unzip the eventon plugin main zip file you downloaed from eventON
Step 2: You will find eventON plugin as zip file, unzip this file as well ( this file has a name like eventON_2.6.14.zip)
Step 3: Log into your FTP client and location folder ../wp-content/plugins/
Step 4: Upload the unzipped eventON folder into the plugins directory in your website
Step 5: go into WP-admin > plugins and activate EventON

*** IMPORTANT UPDATE NOTICES ***

-- from version 2.6.7
There WAS a bug in eventON V2.6.7, that delete all events when updating to latest version via deleting old plugin. If you are on this version and are trying to update to latest version make sure you backup database and files before updating.

More critical steps can be found at: https://www.myeventon.com/news/updating-eventon-2-6-7-important/

-- from version 2.2.8
 
After you update eventON make sure to go to myEventON>settings> appearance and click save changes to re-save the dynamic color changes you have made.
